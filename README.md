### Request

`GET /films`

Returns all films.

### Request

`POST /favorites`

Creates new `favorites` list

### Request

`GET /favorites`

Returns `favorites` lists. Use `?page={}` for pagination and `?search={}` for searching.

### Request

`GET /favorites/:id`

Returns signle `favorites` list.

### Request

`GET /favorites/:id/file`

Returns xlsx file.
